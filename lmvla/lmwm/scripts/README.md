# LMWM Scripts

Planned entrypoints:

- `export_crave_sequences.py`: export CRAVE milestone/state sequences from
  existing CRAVE artifacts.
- `train_state_world_model.py`: train next-milestone and completion-path models.
- `eval_state_world_model.py`: evaluate prediction, planning, and anomaly metrics.
- `export_vla_prompts.py`: export latent milestone subgoals and probability
  prompts for VLA policy integration.
- `summarize_icra_experiments.py`: collect existing graph, training, and runtime
  artifacts into paper-oriented Markdown/CSV/JSON tables.
